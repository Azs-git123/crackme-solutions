# Crackme Solutions

## Deskripsi

Repository ini berisi kumpulan solusi dan writeup dari berbagai challenge crackme yang dikerjakan sebagai bagian dari proses pembelajaran Reverse Engineering.

## Tujuan

* Melatih kemampuan analisis binary.
* Mempelajari teknik static dan dynamic analysis.
* Mendokumentasikan proses penyelesaian challenge secara sistematis.


## Tools yang Digunakan

* Ghidra
* x64dbg
* strings
* Detect It Easy (DIE)
